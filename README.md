## 📖 Descripción del proyecto

Este proyecto es un portafolio web creado con tecnologías web básicas: HTML, CSS y JavaScript. Está diseñado para presentar mi perfil profesional, habilidades, proyectos destacados, y formas de contacto.

### ✨ Características

- Diseño adaptable (responsive)
- Enlaces a proyectos y redes
- Estructura modular y escalable
